Petite description Voici notre projet de TODOLIST
Notre TODOLIST est basique, ajouter et enlever des évènements de telle façon que même lors du rafraichissment de la page WEB nous gardions en mémoire ces évènements.
Pour se faire nous sauvegardons les éléments dans le local storage. Ajout ou suppression agisse en même temps sur la page et le local storage.Vous devez écrire vos événements dans l'encart de bas de page puis le valider, puis, vous pourrez cocher les événements fini et ainsi les supprimer. Bonne utilisation !.
